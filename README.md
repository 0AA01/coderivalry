# CodeRivalry

### 项目介绍

easy 介绍：OJ

hard 介绍：CodeRivalry的目标是做一个高效率OJ，帮助用户在刷题时，集中于刷题本身的知识思维，不受冗余信息干扰，在平台比赛时，保证较高的效率，减少不必要的卡顿，弱化平台的存在，使比赛选手专心于算法题进行思维大战。
